Quick Installation

Get started in minutes

Step 1: Download LinRZ

Download the application file to your computer

wget https://yoursite.com/linrz.py

Step 2: Run LinRZ

Launch the application with GUI interface

python3 linrz.py

Optional: Install Full Format Support

For 7Z and RAR support, install these packages

pip install py7zr rarfile
